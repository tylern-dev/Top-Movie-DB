# Top-Movie-DB
This application was built with React. It uses the [The Movie DB API](https://www.themoviedb.org/documentation/api?language=en) to retrieve popular movies to be displayed.

![home page](https://imgur.com/a/Pj76q)

## To Install:
1. Clone Repo
2. cd into directory
3. `npm install`
4. `npm start`
